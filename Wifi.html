Wifi Function

What is GPS?

What is Wifi and how can we use it to increase GPS efficiency: Wifi is the signal that is produced from a wireless network hub that is projecting an internet connection over radio waves. When a device is connected to the internet it is given an address that represents its location in the internet, this is called an IP address. It is possible to determine the physical location of the wireless access using this address. Thus if a phone can connect to a network, it must be within range of the location that is linked to the IP address of the router. This is preferred over GPS because it allows for use indoors and uses less energy.

What are the limitations of Wifi?

Wifi is not always available and the signal does not stretch far away from the router in most circumstances. This makes Wifi an unreliable source for location data.

How do we retrieve and use the data from the Wifi?

Using several APIs provided by google in the default android library we are able to retrieve location data in the form of latitude and longitude. We enable wifi on the device and begin to push for locations. Because of the speed at which wifi can retrieve a location, the update speed was limited in Googleâ€™s API to around an update every 5 seconds.

How do we implement the Wifi?

Implementation of Wifi used the primary functions built into the API of the LocationManager class and the LocationListener class. Using a button, the application triggers the creation of a LocationManager object. The object is then handed to a DeviceListener class. DeviceListener is a custom class which extends LocationListener to allow for more flexibility and functionality when interacting with the mainActivity.
