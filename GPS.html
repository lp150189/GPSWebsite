GPS

What is GPS?

GPS, or Global Positioning System, is one of the popular and accurate methods of location retrieval on cellular telephones. GPS radio signals from numerous satellites located around the planet to accurately pinpoint a userâ€™s position with huge areas of reception. GPS is one of the most readily available location based systems built into smart phones. This makes GPS the primary location service available on smartphones.

How do we retrieve and use the data from the GPS?

Using several APIs provided by google in the default android library we are able to retrieve location data in the form of latitude and longitude. The device continuously updates the information every time it receives an update from the satellites.

What are the limitations of GPS?

What are the limitations of GPS? There are two main limitations to using GPS. The first is the signal strength: GPS requires near line of sight to the sky to work. Thus when someone enters a building or tunnel, their GPS signal will often be disconnected. Secondly, GPS is extremely taxing on battery life. When compared to other sources of location data, GPS drains almost double the second most taxing service, Wifi.

How do we implement the GPS?

Implementation of GPS used the primary functions built into the API of the LocationManager class and the LocationListener class. Using a button, the application triggers the creation of a LocationManager object. The object is then handed to a DeviceListener class. DeviceListener is a custom class which extends LocationListener to allow for more flexibility and functionality when interacting with the mainActivity. This flexibility Once the DeviceListener is enabled, it will continually receive location updates and set them to a variable for later use.
